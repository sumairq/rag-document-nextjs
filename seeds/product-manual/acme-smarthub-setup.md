# Acme SmartHub X1 — Setup Guide

## What's in the box
The Acme SmartHub X1 package contains the hub unit, a USB-C power adapter (15W),
an Ethernet cable, and a quick-start card. The hub has no internal battery and
must remain plugged in to operate.

## First-time setup
1. Place the SmartHub within 3 meters of your router for initial pairing.
2. Connect the power adapter. The status LED glows solid amber while booting,
   which takes about 45 seconds.
3. When the LED turns slow-blinking blue, the hub is in pairing mode.
4. Open the Acme Home app and tap "Add Device" → "SmartHub X1".
5. Scan the QR code on the bottom of the hub, or enter the 8-digit setup code.
6. Choose your 2.4 GHz Wi-Fi network and enter the password. The SmartHub does
   not support 5 GHz networks for setup, though it can bridge 5 GHz accessories
   afterward.

When setup completes, the LED turns solid white. The hub will automatically
download and install the latest firmware; do not unplug it during this step.

## LED reference
- Solid amber: booting
- Slow-blinking blue: pairing mode
- Solid white: connected and idle
- Fast-blinking white: firmware update in progress
- Solid red: no network connection

## Factory reset
Hold the recessed reset button on the back for 10 seconds until the LED flashes
red three times. A factory reset erases all paired accessories and network
settings, returning the hub to out-of-box state.
